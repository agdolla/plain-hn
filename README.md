# Plain HN

Browse top stories from HackerNews without comments, and without the negativity.

https://plainhn.com or https://news.jorilallo.com

Build using:

- Next.js
- styled-components
- [Interface font](https://rsms.me/interface/)
- Offical HN API

Inspired by [Quiet HN](http://quiethn.com/)

## Dev

```
yarn
yarn dev
```
